## 指南🧭

### 快速上手🚀

**前置：**

- [Node.js](https://nodejs.org)
- [Git](https://git-scm.com/downloads)

```bash
git clone https://github.com/SkanaIS/Mahiro-Project.git
cd Mahiro-Project
npm install
npm run app
```
